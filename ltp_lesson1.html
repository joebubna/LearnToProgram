Learning to Program: Lesson #1

Goal:

To use Javascript for the first time to manipulate a web page. Specifically, we're going to add a 4th item to the list in the Test Result Area.

Steps:

  1. Hit F12 to open developer tools.
  2. Go to the "Console" tab.
  3. Create an <li> object by typing the following into the console:
    var liNode = document.createElement("LI");
  4. Create a Text node by typing:
    var textNode = document.createTextNode("I ROCK!");
  5. Attach the Text node to the LI node by typing:
    liNode.appendChild(textNode); This is creating the DOM (Web Browser representation of a web page) equivalent of the HTML: <li>I ROCK!</li>
  6. Now let's add our new list item to the end of the Test Area list. Type: document.getElementById('testing').children[1].children[1].appendChild(liNode); This first grabs the "testing" section of the document, then the 2nd child element in that section "<span>." It then grabs the 2nd child element under <span> which in this case is our ordered list <ol>. Then appends our newly created list element to the bottom. <section id="testing"> <div id="bar"></div> <span> <h2>Testing Result Area:</h2> <ol> <li>...</li> <li>...</li> </ol> </span> </section>

Test Result Area:

  1. Item 1
  2. Item 2
  3. Item 3

Explanation:

Since text on a page means nothing to a computer, Web browsers convert the HTML text into a data model representation. This underlying representation of the web page is called the DOM (Document Object Model). Think of HTML as the "blueprints" that tell a web browser HOW to construct a page, and the DOM as the actual finished product after the web page has been constructed.

Javascript lets you alter the DOM which opens up the door for all sorts of cool possibilities.

What's next?

This is fairly arduous just to add an extra item to a list. Especially if you have to type it out each time, wouldn't you agree? Do not fear! There are easier ways to manipulate the DOM, this lesson was just about helping you understand how the DOM is separate from the HTML, but also how it follows the parent/child/sibling relationships that indented HTML code shows.

Next we'll learn how to do this task in an easier way by utilizing the power of the Javascript library "jQuery".
